HalfWebLife

Este sitio web nacio como un proyecto academico acerca de la idea de ¿como cambio el mundo en la pandemia del Covid-19? enfocado en el tema de la connectividad y como esta paso a un primer plano de importancia en nuestras vidas en cuarentena y como el sistema no estuvo/esta preparado para asumir la carga.
Bueno si bien esto es cierto, me gustaria saber ¿por que? este sitio trata de responder esa y todas las preguntas que nacen alrededor de este tema y recopila las respuestas que encuentre con el objetivo de que le sirvan a otros y que ellos hagan nuevas preguntas para examinar nuevos aspectos de este tema. Un ciclo de curiosidad e investigacion con el que espero aprender mas acerca de este tema.
